# Passo a Passo de Execução

Esta pasta contém o código do seu agente financeiro.

## Setup do Ollama

````bash
# 1. Instalar Ollama (https://ollama.com/)
# 2. Baixar um modelo leve - Ollama pull Gemma2:2b.
# 3. Testar se funciona: Streamlit run src/app.py
````
## Código Completo

Todo o código-fonte está no arquivo `app.py`.

## Como Rodar

```bash
# 1. Instalar dependências: pip install streamlit pandas requests

# 2. Garantir que Ollama está rodando: Ollama serve

# 3. Rodar a aplicação: streamlit run src/app.py
```
